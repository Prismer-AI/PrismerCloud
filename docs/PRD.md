### 1.1 核心定位重新定义

```

│                    Prismer 产品矩阵 V2                       │
├─────────────────────────────────────────────────────────────┤
```

### 1.2 北极星指标

```
🎯 NSM (North Star Metric):
   每周活跃用户的「研究任务完成数」

   NSM = WAU × Avg(Tasks per User per Week)

   目标: 3 个月内达到 50,000 (WAU: 10K × 5 tasks)
```

### 1.3 核心价值主张

| 平台 | 核心场景 | 独特价值 |
|------|---------|---------|
| **Desktop** | 深度研究、创作输出 | 多窗口 Workspace、完整工具链 |
| **Mobile** | 灵感捕捉、状态监控 | 拍照识别、语音输入、推送提醒 |
| **共通** | AI 研究助手 | 跨设备无缝同步、统一体验 |

## 3. Command - 指挥中心

### 3.1 模块定位

**核心目的**: AI 研究助手的统一入口,整合对话、任务管理、智能推荐

**替代原 Chat + Discovery 模块**,提供更聚焦的体验

### 3.2 信息架构

```
Command
├── AI 对话区 (核心)
│   ├── 对话历史
│   ├── 消息输入
│   └── 快捷指令
├── 任务监控 (可折叠卡片)
│   ├── 进行中的任务
│   ├── 暂停的任务
│   └── 已完成任务
├── 今日推荐 (AI 精选)
│   └── 3-5 篇论文推荐
└── 灵感捕捉 (移动端特有)
    ├── 拍照识别
    ├── 语音输入
    └── 分享导入
```

```typescript

import { invoke } from '@tauri-apps/api/core';
import { readDir, readTextFile, writeTextFile } from '@tauri-apps/plugin-fs';

// 检测运行环境

  return typeof window !== 'undefined' && '__TAURI__' in window;

// 获取平台信息
export const getPlatform = async () => {
```

```typescript
      messages: [],
      tasks: [],

          ...state.messages,
          {
            ...message,
            id: crypto.randomUUID(),
            timestamp: new Date(),
          },
        ],
      })),
      
      addTask: (task) => set((state) => ({
        tasks: [
          ...state.tasks,
          {
            ...task,
            id: crypto.randomUUID(),
            createdAt: new Date(),
          },
        ],
      })),
      
      updateTask: (id, updates) => set((state) => ({
        tasks: state.tasks.map((task) =>
          task.id === id ? { ...task, ...updates } : task
```
